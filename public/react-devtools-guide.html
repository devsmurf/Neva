🔧 React Developer Tools - Yandex Browser Kurulum Rehberi

⚠️ Not: Yandex Browser bazen bazı developer sitelerine erişimde sorun yaşayabilir. Bu normal bir durumdur.

🚀 Yöntem 1: Browser Extension (Önerilen)

1 Yandex Browser'da Extensions sayfasını açın:
browser://extensions/
2 "Geliştirici modu"nu etkinleştirin
3 Chrome Web Store'a gidin ve "React Developer Tools" arayın
4 Extension'ı yükleyin ve etkinleştirin

💻 Yöntem 2: Standalone Kurulum

1 Terminal'de global kurulum yapın:
npm install -g react-devtools
2 DevTools'u başlatın:
react-devtools
3 Projenizin HTML head bölümüne ekleyin:
<script src="http://localhost:8097"></script>

🛠️ Yöntem 3: Geliştirme için Local Kurulum

Projenizdeki package.json'a devtools script'ini ekleyebiliriz:

{ "scripts": { "devtools": "react-devtools", "dev-with-tools": "concurrently \"npm run dev\" \"npm run devtools\"" } }

🌐 Yandex Browser Alternatif Çözümler

1 DNS ayarlarını değiştirin:
  • 8.8.8.8 (Google DNS)
  • 1.1.1.1 (Cloudflare DNS)
2 VPN kullanın (geçici çözüm)
3 Chrome veya Firefox kullanın (alternatif)
✅ Başarı İpucu: React DevTools kurulduktan sonra, browser'ınızın developer tools'unda "Components" ve "Profiler" sekmelerini göreceksiniz.

🔍 Test Etmek İçin

Bu React uygulamanızda DevTools'un çalışıp çalışmadığını test edebilirsiniz:

  1. F12 tuşuna basın (Developer Tools)
  2. "Components" sekmesini arayın
  3. React component ağacını görmelisiniz
🚨 Sorun Devam Ederse:
  • Yandex Browser'ı güncelleyin
  • Cache'i temizleyin
  • Geçici olarak başka bir browser kullanın
